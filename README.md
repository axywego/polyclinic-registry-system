# Polyclinic Registry System
The information system for polyclinic registries was created as a coursework project

### Dependencies
- C++ 23 MinGW
- Qt 6.10.1 or 6.10.2 (I don't f*cking know what I use)
- Docker Compose

### Building on Linux
To build the project in Debug, you need to enter the following commands:
```
cmake  --preset=linux-debug
cmake  --build  --preset=linux-debug-build
./linux-debug/polyclinic_registry_system
```
